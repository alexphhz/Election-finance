# Data 102: Campaign Finance & Voter Support

This repo studies how **state-level economic conditions** relate to **campaign spending** and **voter support**, and which **endorsements** most influence **primary outcomes**. Methods include **multiple hypothesis testing** (FWER/FDR) and **causal inference** (IV + propensity score matching).

## Quick Start
```bash
python -m pip install -r requirements.txt
jupyter lab  # or: jupyter notebook
```
Open `notebooks/analysis.ipynb` and run all cells.

## Data Sources
Raw data are not committed. Use `data/README.md` for links and notes on acquisition/cleaning.
